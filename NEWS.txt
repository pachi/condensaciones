Cambios en la versión
=====================

0.1 (Feb. 2010)
---------------
- Versión inicial

0.2 (Mar. 2010)
---------------
- Edición de las resistencias superficiales
- Uso de materiales resistivos en capas de cerramiento (p.e. cámaras de aire)
- Enlace a la aplicación desde el menú principal [win32]

0.3 (Mar. 2010)
---------------
- Espesor total del cerramiento calculado en informe y lista de capas
- Difusividad y espesor de aire equivalente en capas
- Indica modificación del cerramiento activo
- Maneja capas con resistencia a la difusión del vapor de agua elevado

0.4 (Abr. 2010)
---------------
- Nueva base de datos de materiales en formato .ini
- Nuevo script 'importDB' para importar BBDD de LIDER/CALENER
- Nuevos iconos de aplicación

0.5 (Abr. 2010)
---------------
- Nueva base de datos de cerramientos en formato .ini
- Nueva base de datos de climas en formato .ini
- Base de datos de Materiales más compacta
- Nueva opción de entrada de climas exteriores según base de datos o manual
- Nombre y descripción de cerramientos editables
- Es posible añadir, eliminar y mover cerramientos en la base de datos
- Es posible guardar cerramientos de la sesión activa
- La aplicación se inicia con el primer cerramiento de la base de datos
- La lista de capas incluye el color del material en las gráficas
- Muestra localidad/mes seleccionado en encabezado e informe
- Indica condensaciones superficiales para el mes de enero s/CTE en informe
- Indica condensaciones intersticiales para todos los meses en inform
- Muestra índice de meses con condensaciones intersticiales en informe
- Indica en amarillo si no se cumplen condiciones CTE en otros meses aunque se
  cumplan para las condiciones actuales
- Representa con relleno rojo la zona de condensaciones, en la gráfica P-T
- Salida de informe en navegador HTML
- Documentación en HTML (en web)

0.6 (Dic. 2010)
---------------
- Mejora del rendimiento general de la aplicación
- Inclusión de una barra de condensaciones totales para cada periodo
- El encabezado indica ahora la localidad y periodo activo
- La gráfica de condensación muestra los resultados del periodo activo
- El color asignado a un material se conserva al cambiar el orden de capas
- El informe incluye un histograma de condensaciones acumuladas en cada periodo
- La ayuda en HTML está disponible en el menú de aplicaciones

0.6.1 (May. 2011)
-----------------
- Corrige error en informe HTML con cámaras de aire
- Evita error al eliminar todos los cerramientos de la BBDD
- Evita error en selección de cerramiento activo al eliminar cerramientos
